# Security Policy

The **Awesome AbacatePay** repository is a curated list of resources related to the AbacatePay ecosystem.

Although this repository does not contain production or executable code, security concerns may still arise, especially related to linked resources.

---

## Reporting Security Issues

Please report privately if you identify:

- Malicious or compromised links
- Phishing attempts
- Resources distributing malware
- Projects falsely claiming to be related to AbacatePay
- Accidental exposure of credentials or sensitive information

**Do not open a public issue.**

Report security concerns to:

**security@abacatepay.com**

Or, if available, use **GitHub Security Advisories**.

---

## Responsible Disclosure

We kindly ask that security-related issues are not publicly disclosed until maintainers have had the opportunity to review and address them.

This helps protect users and maintain the integrity of the AbacatePay ecosystem.

---

## Scope Clarification

This security policy applies **only to this awesome list repository**.

For security issues related to:
- Official AbacatePay SDKs
- APIs
- Production services

Please refer to the security policy of the corresponding repository or product.

Thank you for helping keep the AbacatePay open source ecosystem safe and trustworthy.
